# TMS-GIT / Урок 2

Этот репозиторий является учебным и создан для выполнения практических заданий по второму уроку курса, посвященного работе с системой контроля версий Git.

## Структура репозитория

Проект имеет следующую структуру каталогов:

```
tms-git/
└── Lesson-2/          # Каталог, соответствующий второму уроку
    ├── index.html     # Главная HTML-страница проекта
    ├── css/
    │   └── style.css  # Файл каскадных таблиц стилей (CSS)
    └── img/
        └── logo.png   # Логотип или изображение для проекта
```

## Цель проекта

Основная цель этого репозитория и ветки `Lesson-2` — отработка на практике ключевых концепций Git, таких как:
*   Создание и переключение между ветками (`git branch`, `git checkout`).
*   Внесение изменений в файлы проекта.
*   Индексация изменений (`git add`).
*   Фиксация истории изменений с помощью коммитов (`git commit`).
*   Настройка файла `.gitignore` для исключения из контроля версий служебных файлов и каталогов (например, `/.idea`).

## Технологии

*   **HTML** — для структуры веб-страницы.
*   **CSS** — для стилизации веб-страницы.
*   **Git** — для контроля версий и управления исходным кодом.

## Как посмотреть проект

1.  Убедитесь, что на вашем компьютере установлен Git.
2.  Склонируйте репозиторий (или эту конкретную ветку) к себе на машину:
    ```bash
    git clone -b Lesson-2 https://github.com/serjiy/tms-git.git
    ```
3.  Перейдите в каталог проекта:
    ```bash
    cd tms-git
    ```
4.  Откройте файл `index.html` в вашем браузере.

## Настройка Git Ignore

В рамках урока был создан или настроен файл **`.gitignore`**, чтобы исключить из отслеживания Git служебные файлы и папки, специфичные для операционной системы и сред разработки (IDE), такие как:
*   `.idea/` — каталог конфигурации JetBrains IDE (WebStorm, PHPStorm, PyCharm и др.).
*   `.DS_Store` — служебный файл macOS.
*   `Thumbs.db` — служебный файл Windows.

Это лучшая практика для поддержания чистоты репозитория.
